# Веб-приложение на чистом Python + Bootstrap

Простое веб-приложение, созданное без использования фреймворков для понимания основ работы HTTP-серверов, маршрутизации и рендеринга HTML.

## 📋 Описание проекта

Приложение реализует:
- **GET-запросы** — возвращает страницу «Контакты», прочитанную из HTML-файла через `open()`
- **POST-запросы** — принимает данные из формы и выводит их в консоль
- **Bootstrap 5** — все страницы стилизованы с помощью Bootstrap

### Структура страниц
- `Главная` — три карточки товаров с ценой $100 и кнопкой «Купить»
- `Контакты` — форма обратной связи (имя, почта, сообщение) + текст + блок «Профиль/Выход»

## 🛠️ Технологии

- Python 3 (стандартная библиотека, без фреймворков)
- HTML5
- Bootstrap 5 (CDN)
